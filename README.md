# Mäuse 🐭

A minimalist PWA for tracking shared expenses with your partner.

## Features
- Quick expense capture with minimal taps
- Split by percentage or fixed amount
- Monthly overview with totals
- Apple-inspired glass design
- Dark mode support
- Installable as PWA on iPhone

## Install on iPhone
1. Open the app URL in Safari
2. Tap the Share button (⬆️)
3. Select "Add to Home Screen"
4. The app appears on your home screen like a native app

## Tech
- Vanilla HTML/CSS/JS — no frameworks
- IndexedDB for local data persistence
- Service Worker for offline support
- Progressive Web App manifest
